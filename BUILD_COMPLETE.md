## 🚀 DEPLOY IN 3 COMMANDS

### Vercel (Recommended)
```bash
npm install -g vercel
vercel deploy
```
Site live in 60 seconds.

### Docker
```bash
docker build -t avalanche-website .
docker run -p 3000:3000 avalanche-website
```
Self-hosted containerized deployment.

### Node.js
```bash
npm run build
npm start
```
Deploy `node_modules` + `.next` to any Node.js host.
